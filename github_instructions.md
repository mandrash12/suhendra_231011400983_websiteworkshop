
# Requirement 6: GitHub Connection Guide

Follow these steps in your terminal to sync this project to GitHub:

1. **Initialize Git**:
   `git init`

2. **Stage your files** (Note: .gitignore already prevents node_modules):
   `git add .`

3. **First Commit**:
   `git commit -m "Initial commit: Layered architecture for TMS"`

4. **Connect to Cloud**:
   `git remote add origin https://github.com/YOUR_USERNAME/YOUR_REPO_NAME.git`

5. **Push to GitHub**:
   `git push -u origin main`
